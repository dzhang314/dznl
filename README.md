# dznl v0.01

dznl - *David Zhang's Numeric Library* - is a small collection of C++ functions, classes, and templates I maintain for personal use in my numerical work. It is not intended to be a general-purpose toolkit, and is highly tailored to my own use cases, primarily in computational physics and quantum chemistry.

Note that the name of the library is `dznl` (all lowercase), not `DZNL` or `Dznl`. This is for consistency with the C++ convention that namespaces have lowercase names (and so that scientists don't think I'm a national lab).

Recent dznl features include:

- A GitHub repo! (January 2017)
- Class templates for complex and dual numbers (January 2017)
- A 1D Gaussian wavefunction class (January 2017)
